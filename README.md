## :clipboard: About

An calculator to know BBQ expenses easier!


- Less than 6H:
&nbsp;  Meat : 400g /people;
&nbsp;  Beer : 1.2L /people;
&nbsp;  Soda/Water : 1L /people.


- More than 6H
&nbsp;  Meat : 650g /people;
&nbsp;  Beer : 2L /people;
&nbsp;  Soda/Water : 1.5L /people.


- Kids count as half.
